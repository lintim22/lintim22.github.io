Du är på sidan om cpu och jag ska berätta vad CPU egentligen är.
Cpu är även kallat datorns processor.
Det är den som gör så att allt kan kopplas samman och fungera. CPU är egentligen ett kretskort som sitter på moderkortet.
Alla bussar kopplar samman i CPUn.

CPUn sammarbetar med minnet och de kan då koppla samman alla komponenter i datorn så att de kan arbeta med varann. Processorn gör beräkningar samtidigt som den bestämmer över alla andra delar i datorn. CPU eller preocessorn kan även kallas för datorns hjärna. Eftersom att det anses vara där som all information lagras och förs vidare när den behövs. Precis som i människans hjärna.
